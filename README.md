# Debian GUI on Termux (MATE Desktop)

Project ini berisi dokumentasi instalasi *Debian GNU/Linux 13 (Trixie)* di *Termux (Android)* menggunakan *proot-distro, lengkap dengan **MATE Desktop Environment* yang dijalankan melalui *VNC*.

##  Platform
- Android
- Termux (no root)

##  Sistem
- OS: Debian GNU/Linux 13 (Trixie)
- Arsitektur: aarch64
- Desktop Environment: MATE
- Display Server: VNC
- Shell: Bash

## Tools & Teknologi
- Termux
- proot-distro
- Debian
- MATE Desktop
- TightVNC Server
- VNC Viewer
- fastfetch

##  Fitur
- Debian berjalan di Android tanpa root
- Tampilan desktop (GUI)
- Ringan dan cocok untuk belajar Linux
- Bisa digunakan untuk belajar CLI, desktop Linux, dan dasar server
